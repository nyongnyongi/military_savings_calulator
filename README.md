🪖 군 전역 자금 시뮬레이터

대한민국 국군 장병들의 미래 자산 형성 확인을 위한 전역 자금 시뮬레이터입니다.
복무 기간 동안의 월급, 적금, 복리 이자, 그리고 매칭 지원금까지 고려하여
전역 시 예상 자산을 한눈에 확인할 수 있습니다.

📊 주요 기능
-군종 선택 (공군 / 육군·해병대 / 해군)
-월 적금 2개 납입액+금리 커스텀 가능
-복리 기반 적금 만기 금액 계산
-복무 기간 동안의 실수령 월급 계산
-정부 매칭 지원금 자동 반영
-전역 시 예상 총 자산 계산

*반영 기준 (2026년 기준)

본 시뮬레이터는 아래 기준을 기반으로 계산됩니다:

💰 월급 (2026년 기준)
이병 → 일병 → 상병 → 병장 단계별 월급 반영
복무 기간에 따라 자동 적용
📈 적금 이자
사용자 입력 금리를 기준으로 월 복리 계산 적용
🎯 매칭 지원금
납입한 적금 원금과 동일한 금액을 정부 매칭 지원금으로 반영
⚙️ 계산 방식 요약
적금은 월 납입 + 복리 방식으로 계산
총 자산 =
👉 실수령 월급 + 적금 만기 금액(원금+이자) + 매칭 지원금
